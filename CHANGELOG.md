# Changelog — OmniPost

## v1.0.0 — 2026-03-14

### Initial Release

#### Core
- WebSocket backend (Python asyncio)
- Real-time dashboard (HTML/JS/Canvas)
- Smart Copy — formats and copies for each platform, auto-opens app

#### Composer
- Multi-platform toggle (TikTok, Instagram, Facebook, YouTube, Pinterest, Twitter)
- Subtitle field (YouTube title, Facebook hook)
- Text effects: Bold, Italic, CAPS, Emoji boost, Stars, Arrows, Numbered list
- Hashtag input with live suggestions
- Media attachment (images/videos)
- Schedule picker with best-time suggestions

#### Viral Score & A/B Testing
- Real-time viral score 0-100 (emojis, CTA, question, hooks, hashtags, length)
- A/B Test panel — compare two versions, auto-pick winner

#### Visual Creator
- Quote generator: 5 styles × 3 formats → PNG download
- Carousel maker: bullet points → Instagram/LinkedIn slides

#### Hook Library
- 50+ hooks in 8 categories
- Click to insert in composer

#### Content Recycler
- Republish old posts reformatted for 4 platforms
- Claude AI integration for smart reformatting

#### Bio Optimizer + Link in Bio
- AI-generated bios per platform (3 variations)
- Link in Bio page generator (downloadable HTML)

#### Competitor Analyzer
- Full site analysis in 10 seconds
- SEO, performance, 35+ tech detections, security, e-commerce, social
- Traffic estimation, keyword cloud, export report

#### Video Creator
- Slideshow from images + text overlay + music
- Platform-optimized output (TikTok 9:16, YouTube 16:9, etc.)
- FFmpeg integration

#### AI Content Generator
- Claude (Anthropic) integration
- Platform + tone + language selection
- Auto-suggest improvements while typing
- 9 ready-to-use templates

#### Analytics
- Impressions, likes, comments, shares tracking
- Per-platform breakdown + demo data

#### Tips & Best Times
- 19 platform-specific tips
- Real-time best posting times in sidebar
